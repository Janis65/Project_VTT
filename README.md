* Jest to mój pierwszy samodzielny projekt w UE5
* Powstał w celu nauki silnika UE5 i C++
* Moim celem w tym projekcie jest połączenie wolności dostępnej w grach TTRPG z automatyzacją gier (coś pomiędzy Foundry VTT i Baldur's Gate 3)
* Obecnie pracuje nad tym projektem i jest w fazie wczesnego prototypowania
